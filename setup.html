まず先に Git for Windows をインストールしてください。
Claude CodeはGitが必要です。先にインストールしておきましょう。

Git のインストール

  1. git-scm.com を開く
  2. 「Download」をクリック
  3. インストーラを実行 → 全部「Next」でOK
  4. 確認：コマンドプロンプトで git --version を実行
Command Prompt
C:\Users\you> git --version
git version 2.44.0.windows.1

Claude Desktop のインストール

  1. claude.ai/download を開く
  2. 「Download for Windows」をクリック
  3. ダウンロードした Claude.Setup.exe をダブルクリック
「WindowsによってPCが保護されました」と出た場合
「詳細情報」→「実行」をクリックすれば進めます。

よくあるトラブル

困ったときはここをチェック

「WindowsによってPCが保護されました」と出る
これはWindowsの標準的な警告です。
「詳細情報」をクリック → 「実行」を押せば進めます。
Claude Desktopは安全な公式アプリなので、問題ありません。
Macで「xcode-select --install」エラーが出る
ターミナルを開いて、以下のコマンドを実行してください：
xcode-select --install

ポップアップが表示されたら「インストール」をクリック。
完了後にClaude Codeを再起動すれば解決します。
Googleアカウントでログインできない
Googleログインがうまくいかない場合は、
メールアドレスで新規登録し直すのが一番確実です。
Step 1に戻って、メールアドレスで登録してみてください。
「Gitがない」と言われる（Windows）
Step 3の最初に戻って、Git for Windowsをインストールしてください。
git-scm.com → Download → 全部「Next」で OK です。
インストール後、コマンドプロンプトで git --version を実行して確認。
アプリが起動しない / ブラウザが開かない
アプリのログイン画面でURLが表示される場合は、
そのURLをコピーしてブラウザに手動で貼り付けてみてください。
それでも開かない場合は、アプリを一度閉じて再起動を試してください。
それでも解決しない時は？
エラー画面のスクリーンショットを撮って、
あやに送ってください。一緒に解決しましょう！

スクリーンショットの撮り方：
Mac：Cmd + Shift + 4 で範囲選択
Windows：Win + Shift + S で範囲選択
